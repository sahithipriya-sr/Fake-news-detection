#  Fake News Detection System

##  Description

This project builds a Machine Learning model to classify news articles as **Fake** or **Real** using Natural Language Processing (NLP) techniques.
It analyzes textual content and predicts the authenticity of news.

##  Objective

* Detect fake news using ML algorithms
* Understand NLP pipeline
* Build a real-world classification system

##  Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* NLP (TF-IDF / CountVectorizer)

##  How It Works

1. Data collection (Fake & Real news dataset)
2. Text preprocessing (cleaning, removing stopwords)
3. Feature extraction using TF-IDF
4. Model training (Logistic Regression / Naive Bayes)
5. Prediction of news (Fake or Real)

##  Results

* Achieved good accuracy on test data
* Model performs well on unseen news text
